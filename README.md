# [ValoStat](https://rishabh4jakhar.github.io/ValoStat/)
Display Valorant stats of friends

## Features
- Auto update player stats
- Act based rating
- Custom ValoStat Score
- Player vs Player Comparision

## Notes

> Uses [HenrikDev Valorant API](https://api.henrikdev.xyz)

- Valo Stat Score = ACS (30%) + KAST (20%) + KAD (25%) + DD (15%) + WR (10%)
- Favours individual performance over team performance
- Players stats are updated every **4 hours** to stay within API and Github limits
